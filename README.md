- Archlinux
- Visual Studio Code & Clion
- CMake 3.24.1
- g++ (GCC) 12.1.1 20220730

## 项目构建

```bash
git clone --recursive https://github.com/miaohn/hazel.git
```

```bash
sh ./scripts/build.sh
```
